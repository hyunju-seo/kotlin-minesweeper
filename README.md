# kotlin-minesweeper
- 지뢰 찾기를 변형한 프로그램

### 기능 요구사항
- 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- 지뢰는 눈에 잘 띄는 것으로 표기한다.
- 지뢰는 가급적 랜덤에 가깝게 배치한다.
- 각 사각형에 표시될 숫자는 자신을 제외한 주변 8개의 사각형에 포함된 지뢰의 개수다.
